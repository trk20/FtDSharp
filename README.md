# FtDSharp

FtDSharp is a mod for From The Depths that replaces the game's default scripting language (Lua) with C# and makes several enhancements to the scripting API.

# Build Instructions

- Copy dlls from From The Depths "Managed" folder into a "ftd-managed" folder.
- (dotnet) run `FtDSharp.CodeGen` to generate API bindings.
- (dotnet) build the solution.
